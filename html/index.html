How to install in Gmail

  1. Click Copy Signature above.
  2. Open Gmail → Settings (gear icon) → See all settings.
  3. Scroll to the Signature section and click Create new. Name it Punch Signature.
  4. Click inside the signature text box and paste (⌘V on Mac, Ctrl+V on Windows).
  5. Scroll down to Signature defaults and set both For new emails use and On reply/forward use to Punch Signature.
  6. Scroll down and click Save Changes.
